Detailed Description

This PPI contains a set of services to interact with the USB host controller. These interfaces are modeled on the UEFI 2.3 specification protocol EFI_USB2_HOST_CONTROLLER_PROTOCOL. Refer to section 16.1 of the UEFI 2.3 Specification for more information on these interfaces.
