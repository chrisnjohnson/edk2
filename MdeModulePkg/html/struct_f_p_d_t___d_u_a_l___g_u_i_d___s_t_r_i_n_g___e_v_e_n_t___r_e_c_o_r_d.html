UINT16 FPDT_DUAL_GUID_STRING_EVENT_RECORD::ProgressID

ProgressID < 0x10 are reserved for core performance entries. Start measurement point shall have lowered one nibble set to zero and corresponding end points shall have lowered one nibble set to non-zero value; keeping other nibbles same as start point.

CHAR8 FPDT_DUAL_GUID_STRING_EVENT_RECORD::String[0]

ASCII string describing the module. Padding supplied at the end if necessary with null characters (0x00). It is the function name.

UINT64 FPDT_DUAL_GUID_STRING_EVENT_RECORD::Timestamp
