Detailed Description

GUIDs used for ACPI entries in the EFI system table

These GUIDs point the ACPI tables as defined in the ACPI specifications. ACPI 2.0 specification defines the ACPI 2.0 GUID. UEFI 2.0 defines the ACPI 2.0 Table GUID and ACPI Table GUID.
