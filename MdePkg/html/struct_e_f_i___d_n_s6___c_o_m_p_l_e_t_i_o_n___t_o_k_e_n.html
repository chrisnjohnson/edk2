DNS6_ADDR_TO_HOST_DATA* EFI_DNS6_COMPLETION_TOKEN::A2HData

When the Token is used for host address to host name translation, A2HData is a pointer to the DNS6_ADDR_TO_HOST_DATA.

EFI_EVENT EFI_DNS6_COMPLETION_TOKEN::Event

This Event will be signaled after the Status field is updated by the EFI DNSv6 protocol driver. The type of Event must be EFI_NOTIFY_SIGNAL.

DNS6_GENERAL_LOOKUP_DATA* EFI_DNS6_COMPLETION_TOKEN::GLookupData

When the Token is used for a general lookup function, GLookupDATA is a pointer to the DNS6_GENERAL_LOOKUP_DATA.

DNS6_HOST_TO_ADDR_DATA* EFI_DNS6_COMPLETION_TOKEN::H2AData

When the Token is used for host name to address translation, H2AData is a pointer to the DNS6_HOST_TO_ADDR_DATA.

UINT32 EFI_DNS6_COMPLETION_TOKEN::RetryCount

The parameter configured through DNSv6.Configure() interface. Retry number if no response received after RetryInterval.

UINT32 EFI_DNS6_COMPLETION_TOKEN::RetryInterval

The parameter configured through DNSv6.Configure() interface. Minimum interval of retry is 2 seconds. If the retry interval is less than 2 seconds, then use the 2 seconds.

EFI_STATUS EFI_DNS6_COMPLETION_TOKEN::Status

Will be set to one of the following values: EFI_SUCCESS: The host name to address translation completed successfully. EFI_NOT_FOUND: No matching Resource Record (RR) is found. EFI_TIMEOUT: No DNS server reachable, or RetryCount was exhausted without response from all specified DNS servers. EFI_DEVICE_ERROR: An unexpected system or network error occurred. EFI_NO_MEDIA: There was a media error.
