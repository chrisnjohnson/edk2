Detailed Description

The EFI_ISA_ACPI_PROTOCOL provides the services to enumerate and manage ISA controllers on an ISA bus. These services include the ability to initialize, enable, disable, and manage the power state of ISA controllers. It also includes services to query current resources, query possible resources, and assign resources to an ISA controller.
